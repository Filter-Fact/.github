# 🧪 Filter&Facts  
### 과대광고 · 허위광고를 AI로 검증하는 소비자 보호 서비스  

AI 기반 RAG 모델과 MFDS(OpenAPI) 데이터를 활용해  
**건강기능식품·화장품 광고가 허위·과장인지 자동으로 판단해주는 서비스**입니다.

<br>

## 🚀 Tech Stack Overview

| 분야 | 기술 |
|------|------|
| Android | Kotlin, MVVM, Retrofit, Coroutine |
| Backend | Spring Boot, Java 21, JPA, JWT, PostgreSQL, Redis |
| AI / RAG | FastAPI, Python, Nomic-embed-text, Qwen 2.5 1.5B, pgvector |
| DevOps | Naver Cloud NKS, Kubernetes, Docker, GitHub Actions CI/CD |

## 🧠 Service Summary

Filter&Facts는 다음 AI 기반 기능을 제공합니다:

- ✔ **광고 문구 자동 분석** (LLM 기반)
- ✔ **건강기능식품·화장품 효능 검증**
- ✔ **MFDS 고시 기준 매칭**
- ✔ **허위·과장 가능성 자동 판단**
- ✔ **과거 검색 히스토리 기반 개인화된 결과 제공**
